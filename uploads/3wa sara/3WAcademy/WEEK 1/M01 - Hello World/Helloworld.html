Au cours de catte formation nous allons avoir 2 matiéres:

L'intégration

Le HTML

le HTML sert à créer le contenu du site: elle va contenir uniquement la partie textuelle du site.

Le CSS:

Le CSS va nous permettre d'ajouter la couche graphique à nous site: les couleurs, les formes, le fond, etc.

Le dévloppement

Le javascript:

javascript est un langage de programmation de script employé pour rendre une page web interactive, le comporetement de la page et son apparence pourravarier en fonction de certains paramétres

Le PHP:

Le php est un language de programmation libre, utilisé pour généré des pages web dynamiques, qui sont donc modifés en fonction de certaines informations (heures,nom del'utilisateur,formulaire rempli par l'utilisateur etc).

MySQL

MySQL est un systéme de gestion de bases de données, il permet d'enregistrer des informations rempli par l'utilisateur (exemple: formulaire de céation de compte) pour ensuite être ré-utilisé au moment approprié (ex:connexion à un site internet avec identifiant et mot de passe).
